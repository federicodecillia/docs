---
title: Getting a GPT Chatbot API Key
---

The following guide walks you through how to obtain an API key. If you already have your API key,
you can also explore practical examples and integration tips to jump start your journey with GPT Chatbot. Let’s begin!

## Step 1: Access GPT Chatbot Platform

Go to the [GPT Chatbot platform](https://app.gptchatbot.it).

## Step 2: Create an Account (If you don't already have one)

If you don’t have an account on the GPT Chatbot platform, click on the “Sign Up” or “Create Account”
button and follow the registration process. Provide the required information to create your account.

## Step 3: Access Your Account Menu

After successfully logging in, you’ll be directed to the app. Look for the “Account” menu in the platform’s
navigation bar. Click on it to access your account settings.

<Frame>
  <img src="/images/api-key-1.png" />
</Frame>

## Step 4: Navigate to API Keys Section

Inside your account settings, you will find an “API Keys” section. Within the API Keys section,
you should see an option to “Generate new key”. Click on it to initiate the process of creating a new API key.

<Frame>
  <img src="/images/api-key-2.png" />
</Frame>

## Step 5: Enter Key Description and Generate

In the "Create New API Key" dialog, provide a description for your API key to help identify its purpose.
Once you’ve entered the description, click the “Generate” button to create your API key.

<Warning>
  Be sure to copy and save your API key in a secure location. Do not share it
  with anyone, as it provides access to the GPT Chatbot API services. Treat your
  API key like a password.
</Warning>

<Frame>
  <img src="/images/api-key-3.png" />
</Frame>

Now, you’ve successfully requested, generated, and saved your GPT Chatbot API key.
You can use this key to authenticate your requests and integrate GPT Chatbot into your applications, scripts, or projects.

If you’re ready to dive further and explore practical examples of using the GPT Chatbot API, please visit the **"Live Examples"** section on the left sidebar.
